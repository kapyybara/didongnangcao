# CÁCH SỬ DỤNG 

* Copy file example.env -> .env và update database 
* Tiếp theo tải package và chạy directus
    ```
    cd services/directus 
    yarn install 
    yarn start
    ```
* Tương tự với mainapp 
    ```
    cd services/mainapp 
    yarn install 
    yarn start
    ```
Note:  gõ lệnh a trong mainapp để chạy trên android
